# إعداد حساب الإدمن

## إنشاء حساب إدمن

لإنشاء حساب إدمن، استخدم الأمر التالي في Terminal:

```bash
php artisan tinker
```

ثم في Tinker، نفذ الأوامر التالية:

```php
use App\Models\User;
use Illuminate\Support\Facades\Hash;

User::create([
    'name' => 'Admin',
    'phone' => '+963-11-7654321', // استبدل برقمك
    'password' => Hash::make('password'), // استبدل بكلمة مرور قوية
    'user_type' => 'admin',
    'is_verified' => true,
]);
```

## معلومات تسجيل الدخول

- **الرقم**: الرقم الذي أدخلته أعلاه
- **OTP**: `12345` (رقم افتراضي للتطوير)

## ملاحظات أمنية

1. **لا تشارك** معلومات حساب الإدمن مع أي شخص
2. **غيّر الرقم** في الإنتاج إلى رقم حقيقي
3. **فعّل إرسال SMS** في الإنتاج بدلاً من OTP الافتراضي
4. **استخدم كلمة مرور قوية** لحساب الإدمن

## الوصول للداشبورد

بعد إنشاء الحساب، يمكنك الوصول للداشبورد عبر:
- `/login` - تسجيل دخول عادي (يستخدم نفس صفحة تسجيل الدخول)
- `/admin/dashboard` - الداشبورد الرئيسي (بعد تسجيل الدخول كإدمن)
- `/admin/users` - إدارة المستخدمين

## إدارة المستخدمين

من خلال قسم "Users" في الداشبورد، يمكنك:
- عرض جميع المستخدمين
- إضافة مستخدمين جدد
- تعديل بيانات المستخدمين
- حذف المستخدمين (عدا الإدمن)
- تصفية المستخدمين حسب النوع والحالة

## أنواع المستخدمين

1. **Customer** - عميل عادي
2. **Store Owner** - صاحب متجر
3. **Driver** - سائق توصيل
4. **Admin** - إدمن (لا يمكن حذفه، ولا يمكن إنشاؤه من الواجهة)

## نظام OTP

- **رقم افتراضي**: `12345`
- **مدة الصلاحية**: 5 دقائق
- **إعادة الإرسال**: متاحة
- **التطوير**: يتم تسجيل OTP في logs
- **الإنتاج**: يجب تفعيل إرسال SMS حقيقي
